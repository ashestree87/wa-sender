Convert SVG to ICO

This page helps you convert the SVG favicon to ICO format:

  1. The SVG is rendered below at different sizes
  2. Right-click on each canvas and select "Save image as..."
  3. Save them as PNG files
  4. Use an online converter like Convertio to convert the PNG to ICO
  5. Replace the existing favicon.ico in the public folder
